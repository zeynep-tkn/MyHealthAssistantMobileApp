# My Health Assistant
## Mobile App 
### Android Studio-Java 

My Health Assistant, fiziksel ve zihinsel sağlığınızı desteklemek için geliştirilmiş, yapay zekâ entegreli bir mobil uygulamadır.

## Özellikler

- Sağlık AI asistanı
- Beslenme programları
- Spor programları
- Eğitim içerikleri
- Motivasyon bölümü
